# MackySoft.Ucli.Infrastructure

Shared infrastructure services for uCLI runtime components.

## What This Package Provides

`MackySoft.Ucli.Infrastructure` contains shared infrastructure code used by uCLI components that run outside the pure contract boundary.

It includes:

- Filesystem and storage path helpers
- Process liveness probing
- Project and index fingerprint helpers
- Transport path utilities
